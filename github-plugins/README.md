# NMP 插件仓库

这是 NMP (Network Monitoring Platform) 的官方插件仓库。

## 可用插件

| 插件名称 | 版本 | 描述 |
|---------|------|------|
| system-backup | 1.0.0 | 系统备份与还原 - 支持数据库和配置文件的完整备份 |

## 如何使用

1. 在 NMP 管理界面中，点击侧边栏的"插件中心"
2. 在插件市场中浏览可用插件
3. 点击"安装"按钮即可自动下载安装

## 开发插件

如果你想开发自己的插件，请参考 [插件开发文档](https://github.com/Erion6677/nmp-platform/blob/main/docs/plugin-development.md)
