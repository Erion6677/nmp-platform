## 系统概述

NMP Platform（网络监控平台）是一个基于Web的网络设备监控系统，支持对路由器、交换机、服务器等网络设备进行实时监控和历史数据分析。

### 主要功能
- **设备管理**: 添加、配置和管理网络设备
- **实时监控**: 查看设备的实时状态和性能指标
- **历史数据**: 分析设备的历史性能趋势
- **告警管理**: 配置和管理设备告警规则
- **用户权限**: 多用户和角色权限管理
- **数据导出**: 导出监控数据和报表

### 支持的设备类型
- RouterOS设备（MikroTik路由器）
- Linux服务器
- 支持SNMP的网络设备
- 其他通过SSH访问的设备

## 登录和认证

### 首次登录

1. 打开浏览器，访问系统地址（如：`http://192.168.1.100`）
2. 在登录页面输入用户名和密码
3. 首次安装后，使用安装时创建的管理员账户登录

### 登录界面

![登录界面](images/login.png)

- **用户名**: 输入您的用户名
- **密码**: 输入您的密码
- **记住我**: 勾选后下次访问无需重新登录
- **忘记密码**: 点击重置密码（需要管理员配置邮件服务）

### 修改密码

登录后可以修改密码：
1. 点击右上角用户头像
2. 选择"个人设置"
3. 在"安全设置"中修改密码

## 设备管理

### 添加设备

1. 进入"设备管理" → "设备列表"
2. 点击"添加设备"按钮
3. 填写设备信息：

   **基本信息**
   - 设备名称：设备的显示名称
   - 设备类型：选择设备类型（RouterOS、Linux等）
   - IP地址：设备的管理IP地址
   - 端口：SSH端口（默认22）

   **认证信息**
   - 用户名：SSH登录用户名
   - 密码：SSH登录密码
   - 或选择SSH密钥认证

   **其他设置**
   - 设备分组：选择或创建设备分组
   - 标签：添加设备标签便于管理
   - 描述：设备的详细描述

4. 点击"测试连接"验证设备连接
5. 点击"保存"完成添加

### 设备列表

设备列表显示所有已添加的设备：

- **设备状态**: 在线/离线状态指示
- **基本信息**: 设备名称、IP地址、类型
- **最后更新**: 最后一次数据更新时间
- **操作按钮**: 编辑、删除、查看详情

### 设备详情

点击设备名称进入设备详情页面：

1. **概览信息**
   - 设备基本信息
   - 当前状态和性能指标
   - 系统信息

2. **接口列表**
   - 网络接口信息
   - 接口状态和流量统计
   - 接口配置

3. **监控图表**
   - CPU使用率
   - 内存使用率
   - 网络流量
   - 自定义指标

### 设备分组

设备分组用于组织和管理大量设备：

1. 进入"设备管理" → "设备分组"
2. 点击"创建分组"
3. 输入分组名称和描述
4. 将设备拖拽到相应分组

### 批量操作

支持对多个设备进行批量操作：
1. 在设备列表中勾选多个设备
2. 点击"批量操作"按钮
3. 选择操作类型（删除、移动分组、添加标签等）

## 监控功能

### 实时监控

实时监控显示设备的当前状态：

1. 进入"监控中心" → "实时监控"
2. 选择要监控的设备或设备组
3. 查看实时数据：
   - CPU使用率
   - 内存使用率
   - 网络流量
   - 磁盘使用率
   - 系统负载

### 监控仪表板

仪表板提供设备监控的总览：

1. **系统概览**
   - 设备总数和在线状态
   - 告警统计
   - 性能趋势

2. **设备状态**
   - 设备健康状态
   - 性能排行榜
   - 异常设备列表

3. **网络拓扑**
   - 网络设备连接关系
   - 链路状态和流量
   - 故障点标识

### 自定义监控

可以为特定设备配置自定义监控项：

1. 进入设备详情页面
2. 点击"监控配置"
3. 添加自定义监控项：
   - 监控指标名称
   - 数据采集方式
   - 采集频率
   - 告警阈值

## 数据查询

### 历史数据查询

查询设备的历史监控数据：

1. 进入"数据分析" → "历史查询"
2. 选择查询条件：
   - 设备：选择要查询的设备
   - 时间范围：选择查询的时间段
   - 指标：选择要查询的监控指标
   - 聚合方式：平均值、最大值、最小值等

3. 点击"查询"获取数据
4. 查看结果：
   - 图表显示
   - 数据表格
   - 统计信息

### 数据导出

支持将监控数据导出为多种格式：

1. 在查询结果页面点击"导出"
2. 选择导出格式：
   - CSV格式
   - Excel格式
   - PDF报表
   - 图片格式

3. 设置导出选项
4. 下载导出文件

### 报表生成

系统支持生成定期报表：

1. 进入"数据分析" → "报表管理"
2. 创建报表模板：
   - 报表名称和描述
   - 数据源和指标
   - 报表格式和样式
   - 生成频率

3. 配置报表发送：
   - 接收人邮箱
   - 发送时间
   - 发送频率

## 用户管理

### 用户账户管理

管理员可以管理系统用户：

1. 进入"系统管理" → "用户管理"
2. 查看用户列表
3. 添加新用户：
   - 用户名和密码
   - 邮箱地址
   - 用户角色
   - 权限设置

### 角色权限管理

系统支持基于角色的权限控制：

1. **预定义角色**
   - 超级管理员：所有权限
   - 系统管理员：系统管理权限
   - 设备管理员：设备管理权限
   - 普通用户：只读权限

2. **自定义角色**
   - 创建自定义角色
   - 分配具体权限
   - 权限包括：设备管理、监控查看、系统配置等

### 用户组管理

可以创建用户组来批量管理用户权限：

1. 创建用户组
2. 添加用户到组
3. 为用户组分配权限
4. 用户自动继承组权限

## 常见操作

### 快速添加设备

1. 点击顶部导航的"+"按钮
2. 选择"添加设备"
3. 快速填写必要信息
4. 保存并开始监控

### 查看设备状态

1. 在仪表板查看设备概览
2. 点击设备名称查看详情
3. 查看实时监控数据
4. 检查告警信息

### 处理告警

1. 查看告警列表
2. 点击告警查看详情
3. 确认或忽略告警
4. 添加处理备注

### 生成报表

1. 选择报表模板
2. 设置时间范围
3. 选择设备和指标
4. 生成并下载报表

### 搜索功能

系统提供全局搜索功能：
1. 在顶部搜索框输入关键词
2. 搜索范围包括：
   - 设备名称和IP
   - 告警信息
   - 用户和角色
   - 配置项

### 快捷键

系统支持常用快捷键：
- `Ctrl + /`：显示快捷键帮助
- `Ctrl + K`：打开全局搜索
- `Ctrl + Shift + D`：切换深色主题
- `F5`：刷新当前页面

## 故障排除

### 常见问题

1. **无法登录**
   - 检查用户名和密码
   - 确认账户未被锁定
   - 联系管理员重置密码

2. **设备显示离线**
   - 检查设备网络连接
   - 验证SSH连接信息
   - 查看设备日志

3. **数据不更新**
   - 检查设备连接状态
   - 确认监控服务运行正常
   - 查看系统日志

4. **页面加载慢**
   - 检查网络连接
   - 清除浏览器缓存
   - 联系管理员检查服务器性能

### 获取帮助

如需帮助，可以：
1. 查看在线帮助文档
2. 联系系统管理员
3. 提交技术支持请求
4. 查看系统日志获取错误信息
